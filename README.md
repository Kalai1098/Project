##Digital Prescription Record Analysis - P1

#Project Description: 
Develop a platform or tools for managing, tracking, and analyzing electronic medical prescriptions, improving record-keeping, patient safety, and process efficiency for healthcare providers.

#Objective:
· Digitize prescription management to reduce errors.

· Real-time tracking and secure, accessible records.

· Analyze prescription data for trends and quality monitoring.

#Tools Used:
· Python (Pandas, Seaborn, Matplotlib)

· SQL for data management

#Project Type: 
Healthcare Informatics

#Outcome:
A robust Python-based CLI tool that automates the ingestion, cleaning, and storage of healthcare data, enabling users to run predefined analytical queries and get insights directly from the terminal
